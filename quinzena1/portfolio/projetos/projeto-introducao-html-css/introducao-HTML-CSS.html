Fundamentos de HTML

Acesse a aula da Khan Academy clicando aqui

Documentos HTML(Hypertext Markup Language, ou “Linguagem de Marcação em Hipertexto”) são escritos por tags, que são cada conteúdo dentro dos "<>".

A aula inicia com a visualização de uma página web em branco que será construída no decorrer do curso. Para isso nos são apresentados algumas propriedades do esqueleto da página:

<!DOCTYPE html>:

Identifica para o navegador que a página está escrita no novo formato HTML.

<html> e </html>:

Respectivamente: abre e fecha o documento HTML. Dentro desta tag ficará toda a programação da página.

<head> e </head>:

Ficam dentro desta tag as instruções para que o navegador leia a página. Como as cores e tipo de letras usadas no documento.

<meta charset="utf-8">:

Localizada dentro da tag <head>, a tag <meta charset=""> identifica qual o conjunto de caracteres será usado na codificação deste documento, enquanto a propriedade "utf-8" diz ao navegador que serão os caracteres mais simples, como os presentes na língua inglesa.

<title> e </title>:

Define o título da página no navegador.

<body> e </body>:

Esta é a seção visível do site para o usuário. Dentro dela fica todo o conteúdo visível como títulos, textos, imagens, links, etc. Ou seja, tudo o que se pode ver nesta página está dentro desta tag.

<h1> e </h1>:

Esta tag define os títulos e seus tamanhos. Indo do <h1>, que é o maior título e diminuindo até o <h6>, menor título. Exemplos:

<h1> </h1>

<h2> </h2>

<h3> </h3>

<h4> </h4>

<h5> </h5>
<h6> </h6>

<p> e </p>:

Dentro desta tag ficam os parágrafos simples do documento.

<br>:

O navegador não processa as quebras de linhas digitadas no campo de programação, então para que haja a quebra, precisa-se usar esta tag no final da linha, para que haja quebra. Exemplo:
Na codificação:
<p> Para que aconteça esta quebra, <br> precisa-se deste símbolo <br> para o navegador processá-la </p>
Na visualização:
Para que aconteça esta quebra,
precisa-se deste símbolo

HTML: Listas

Existem dois tipos de lista dentro do HTML: Listas ordenadas, com os itens iniciados por números e listas não ordenadas, com os itens iniciados com símbolos.

<ul> e </ul>:

Lista não numerada.

<ol> e </ol>:

Lista numerada. Obs: ao trocar a ordem de qualquer item da lista, ela automaticamente reordena o número inicial.

<li> e </li>:

Cada item da lista deve ficar nesta tag.

Exemplos de listas:

HTML: Imagens

Acesse a aula da Khan Academy clicando aqui

Para adicionar uma imagem ao seu documento HTML você vai precisar da url desta imagem.

<img src="url da imagem desejada"> :

Insira dentro desta tag, entre as áspas, o endereço da imagem desejada.
Exemplo:

Código:
<img src="https://c.pxhere.com/photos/98/63/dog_black_puppy_cute_happy_young_doggy_fun-552464.jpg!d">
Visualização:

Imagem retirada de: https://pxhere.com/

<img src="url da imagem desejada" alt = "Descrição da imagem"> :

O atributo alt= permite que se descreva a imagem inserida para o caso de ela não estar mais disponível na hora de acesso ou mesmo para os aplicativos de leitura automáticos de texto, comumente usados pela população cega, aumentando a acessibilidade da página.

<img src="url da imagem desejada" width="largura desejada para a imagem" height="altura desejada para a imagem" > :

Os atributos "width" e "height" permitem, respectivamente, que você escolha a altura e largura da sua imagem. Porém quando informa-se apenas um deles, o computador calcula automaticamente o outro atributo, de forma proporcional, evitando assim que a imagem fique distorcida.

Código:

Usando os dois atributos:
<img src="https://www.kasandbox.org/programming-images/animals/rabbit.png" width="300" height= "100">

Código:

Usando apenas um dos atributos:

Introdução a CSS

Acesse a aula da Khan Academy clicando aqui

CSS é usado no HTML para a estilização da página.
* Qualquer elemento do HTML pode ser selecionado com o CSS, basta utilizá-los sem os "< >"

<style> e </style>:

Esta tag fica dentro da tag <head> e nela serão colocados os itens de estilização da página, como cores de títulos, subtítulos, parágrafos ou fundo.

Exemplo:
Para mudar a cor de todos os títulos h1 do seu documento:
<head>
<style>
h1{
color:red;
}
</style>
</head>

Assim todos os títulos h1 estarão na cor selecionada, a menos que haja alguma regra específica. O mesmo vale para parágrafos, tabelas, listas, etc.

As cores usadas em CSS são em RGB ("red, green, blue", ou "vermelho, verde e azul". Pode-se usar os comandos "red", "orange", "green", etc., mas, para cores mais específicas, usar:
color: rgb(255, 0, 0); (esta proporção resulta em vermelho).

CSS: Seleção por id

Acesse a aula da Khan Academy clicando aqui

Seleciona uma única tag.
O identificador deve ser único, cuidando para que não se repita em outro item, pois isso confundirá o comando dado ao computador.

Exemplo:

<body>
<p id="identificador-do-paragrafo"> Texto do parágrafo seleção por id. </p>
</body>

Cria na tag body um id para este parágrafo, então pode selecioná-lo na tag style da seguinte forma:

<style>
#identificador-do-paragrafo{
background-color: yellow;
}
</style>

Visualização:

Texto do parágrafo seleção por id.

Neste exemplo a cor de fundo do páragrafo identificado como "identificador-do-paragrafo" para a cor amarela.

    Notas:
  • Para chamar o id, sempre usar "#".
  • O id sempre deve começar com uma letra e somente pode ter caracteres comuns (sem acentos) e "-" ou "_".

CSS: Seleção por classe

Acesse a aula da Khan Academy clicando aqui

Usa-se a seleção por classe para selecionar mais de um item que deva ter as mesmas características no documento, neste documento usei a seleção por classe para que todas as tags, seleções e itens novos que aparecem no texto ficassem destacados em negrito e da cor vermelha.

Exemplo:

<body>
<p class="identificador-da-classe"> Texto do parágrafo seleção por classe. </p>
<p class="identificador-da-classe"> Segundo texto do parágrafo seleção por classe. </p>
</body>

Cria na tag body uma classe para este e demais parágrafos que devam ter as mesmas propriedades como tamanho, cor, estilo, etc., então pode selecioná-la na tag style da seguinte forma:

<style>
.identificador-da-classe{
color:rgb(0, 255, 251);
}
</style>

Tabelas HTML

Acesse a aula da Khan Academy clicando aqui

<table> e </table>:

Tags de abertura e fechamento, respectivamente, da tabela.

<thead> e </thead>:

Abertura e fechamento do cabeçalho da tabela. Usado somente no cabeçalho.

<tr> e </tr>:

Abertura e fechamento de cada linha da tabela.

<th> e </th>:

Cada célula do cabeçalho da tabela. Usado somente dentro da tag <tr> no cabeçalho da tabela.

<tbody> e </tbody>:

Abertura e fechamento do corpo da tabela.

<td> e </td>:

Abre e fecha cada célula do corpo da tabela. Usar dentro da tag de linha.

Exemplo de tabela (Código):
<table>
<thead>
<tr>
<th> Itens </th>
<th> Quantidade </th>
</tr>
</thead>
<tbody>
<tr>
<td> Chocolate </td>
<td> 2 barras </td>
</tr>
<tr>
<td> Leite </td>
<td> 12 caixas </td>
</tr>
</tbody>
</table>

Exemplo de tabela (Visualização):

Itens Quantidade
Chocolate 2 barras
Leite 12 caixas

Existem diversas propriedades de textoEstas propriedades ficam dentro dos seletores de CSS, dentro tag style.

    Exemplos de mais propriedades de texto:
  • font-size: define o tamanho da fonte. Usa-se como unidade, mais comumente "em", para proporção e "px" para tamanho em pixel.
  • font-weight: define a espessura da fonte, como negrito (bold).
  • line-height: define a altura(height) da linha(line).
  • text-align: define o alinhamento do texto - center/right/justified.
  • font-family: define a família da fonte ou a própria fonte a ser usada.
  • Para mais propriedades, procurar o que deseja na internet,
  • Pode-se abreviar as propriedades de fonte com: font: weight/style size family;
    Mais informações sobre font-family:
    Famílias de fontes:
  • Quando coloca-se apenas o nome da família, o computador seleciona a sua fonte principal na família.
  • serif - fontes com prolongamentos nas extremidades, como a Times New Roman.
  • sans-serif : fontes mais "retas", sem prolongamentos nas extremidades, como a Arial.
  • cursive : simula fontes escritas à mão.
  • fantasy : fontes decorativas.
  • Arial (família sans-serif)
  • Para fontes mais específicas colocar o nome da fonte.
    • Exemplos de fontes:
  • Arial (família sans-serif)
  • Helvetica (família sans-serif)
  • Verdana (família sans-serif)
  • Times New Roman (família serif)
  • Georgia (família serif)
  • Courier (família monospace)
  • Andale Mono (família monospace)
  • Comic Sans MS (família cursive)
  • Apple Chancery (família cursive)
  • Snell Roundhand (família cursive)
  • Impact (família fantasy)
  • Jazz LET (família fantasy)
  • Tratatello (família fantasy)
    • Observações a respeito do uso do nome específico da fonte:
  • Eventualmente o navegador não possui a determinada fonte escolhida, como garantia insira outra fonte reserva e/ou o nome da família, caso contrário o navegador irá ignorar o chamado pelo nome da fonte e usará a fonte padrão do HTML. Ao fazer esta lista de fontes, meu navegador não possuía algumas das fontes usadas no exemplo, portanto nestas, para mim, aparece a fonte Times New Roman em todas os exemplos de fontes que ele não possui (talvez aparecerá assim para você também).
  • Para que isso não aconteça no documento, insira a fonte reserva e/ou família da seguinte forma: font-family:"nome da fonte desejada", "nome da fonte reserva caso haja", "nome da família";
    Por exemplo: para selecionar primeiramente a Helvética, como segunda opção a Arial e então ,como terceira opção, ficaria da seguinte forma: font-family: "helvética", "arial", "sans-seriff"; .

Herança CSS

Acesse a aula da Khan Academy clicando aqui

Algumas propriedades de classe são herdadas das classes mães passando tais características para as classes filhas, só deixando de ser utilizadas caso uma regra mais específica seja acionada.

Como exemplo, neste documento, todos os h2 de título têm a configuração de espaçamento "5em" e sublinhado, exceto o h2 presente no exemplo do capítulo "Fundamentos de HTML", que mantém o espaçamento em "1em" por uma regra mais específica da classe em que está presente, porém mantém a cor definida para a tag h2 por herdar esta propriedade. Assim como o título inicial, h1, do documento, que está com a configuração para espaçamento de linha em "3em" e alinhamento centralizado, ambas as propriedades não são passadas ao títilo h1 presente no exemplo do mesmo capítulo, ambos exemplos pertencem à mesma classe.
